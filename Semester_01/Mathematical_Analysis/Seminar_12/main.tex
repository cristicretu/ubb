\( A \in \mathbb{R}^{m \times n} \) is a full rank matrix with \( m < n \). 

\subsection*{1}

We prove that \( AA^T \) is positive semi-definite using the definition: \(x^T AA^T x \geq 0.\), where \( x \in \mathbb{R}^m \).

\( A \) has full rank \( \Rightarrow Ax \neq 0 \) for all \( x \neq 0 \)  \( \Rightarrow x^T AA^T x = (Ax)^T (Ax) = \| Ax \|^2 \geq 0 \Rightarrow x^T AA^T x \geq 0 \)

We prove that \( AA^T \) is positive definite, \( x^T AA^T x > 0. \) using the same argument as before for \( x \neq 0 \), \( Ax \neq 0 \) since \( A \) has full rank \( \Rightarrow x^T AA^T x = \| Ax \|^2 > 0  \Rightarrow \) \( AA^T \) is positive definite.

A positive definite matrix is invertible. Since \( AA^T \) is positive definite \( \Rightarrow AA^T \) is also invertible.
    
    \subsection*{2}

\begin{equation*}
\min_{x \in \mathbb{R}^n} \| x \|^2 \quad \text{subject to} \quad Ax = b.
\end{equation*}

\[ f(x) = \| x\|^2 = x^Tx, g(x) = Ax - b\]
\[L(x, \lambda) = x^Tx - \lambda^T (Ax - b)\]

\begin{align*}
\frac{\partial L}{\partial x} &= 2x - A^T\lambda = 0 \\
\frac{\partial L}{\partial \lambda} &= Ax - b = 0
\end{align*}

\begin{align*}
x = \frac{1}{2}A^T\lambda \\
Ax = b
\end{align*}

\[ \Rightarrow \frac{1}{2}AA^T\lambda = b\]

Since \( AA^T \) is invertible (from part 1), we can solve for \( \lambda \), without the \(\frac{1}{2}\) factor:
\begin{equation*}
\lambda = (AA^T)^{-1}b.
\end{equation*}

Substituting back for x \( \Rightarrow x^* = A^T\lambda = A^T(AA^T)^{-1}b \)

\subsection*{3}

Since \(x\) and \(x^*\) are both solutions for \(Ax = b \Rightarrow Ax = Ax^* = b\).
We substitute \(\Rightarrow A(x - x^*) = 0\). THis means that \((x-x^*)^Tx^* = 0.\)

Also, since \(x^*\) minimizes \( \| x \|^2 \) and is a unique solution for the min. norm this means that \( \| x^* \| \leq \| x \|. \)

\subsection*{4}

\[
\min_{x \in \mathbb{R}^n} \left( \|Ax - b\|^2 + \alpha \|x\|^2 \right )
\]

We calculate the derivative with respect to x, and compare it with 0 to find the minimum of the function:

\[
f(x) = \|Ax - b\|^2 + \alpha \|x\|^2 = (Ax - b)^T(Ax - b) + \alpha x^T x.
\]


\[
\frac{\partial f}{\partial x} 2A^T(Ax - b) + 2\alpha x = 0.
\]

\[
A^T Ax + \alpha x = A^T b.
\]

\[
(A^T A + \alpha I)x = A^T b.
\]

\[ \Rightarrow x_{\alpha} = (A^T A + \alpha I)^{-1} A^T b\]

\subsection*{5}

\( A^TA + \alpha I \) is positive definite if:

\[
x^T(A^TA + \alpha I)x > 0.
\]

\[
x^T(A^TA + \alpha I)x = x^T A^T A x + \alpha x^T x = (Ax)^T (Ax) + \alpha x^T x = \| Ax \|^2 + \alpha \| x \|^2.
\]

\( \| Ax \|^2 \geq 0 \) and \( \alpha \| x \|^2 > 0 \) \( \Rightarrow A^TA + \alpha I \) is positive definite. This also results that \( A^TA + \alpha I \) is invertible.


\[
x_{\alpha} = (A^TA + \alpha I)^{-1}A^Tb.
\]

We apply the limit to both sides: as \( \alpha \) approaches 0, \(\alpha I\) approaches 0 as well.

\[
\lim_{\alpha \to 0} x_{\alpha} = (A^TA)^{-1}A^Tb = x^*.
\]
